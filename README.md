## `Running locally:`
 - Access the folder *database* 
 - Run *bash ./create-databse*
 - Configure *.env* file

# 

## `.env SETUP`

### DATABASE_URL
 - postgres://user:password@host:port/valex

### PORT=port number
 - Default 5000

### CRYPTR=key
 - Your super secret key

### BCRYPT=salts
 - Number of salts, e.g. 10

### MODE=DEV
